# Cheatsheet

## Files

| Action | Say |
|--------|-----|
| Read | `Show me config.json` |
| Create | `Create utils.py with a function that...` |
| Edit | `In utils.py, change X to Y` |
| Find | `Find files containing "password"` |
| Delete | `Delete the temp.log file` |

## Commands

| Action | Say |
|--------|-----|
| Run code | `Run main.py` |
| Install | `Install pandas` |
| Test | `Run the tests` |
| Build | `Build the project` |
| Git status | `Show git status` |
| Commit | `Commit with message "Add feature"` |

## Patterns

**Be specific:**
```
Create a function that validates emails and returns true/false
```

**Iterate:**
```
That's not handling empty strings. Fix it.
```

**Debug:**
```
I got this error: [paste]. Fix it.
```

**Explain:**
```
What does this function do?
```

**Extend:**
```
Add logging to this script
```

## Built-in Commands

| Command | Purpose |
|---------|---------|
| `/help` | Show help |
| `/clear` | Fresh start |
| `/cost` | Usage stats |
